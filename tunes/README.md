# Music files for Heli Rescue

The game plays background music from VGZ files (compressed VGM register data
for the YM3812/OPL2 chip). Place any `.vgz` files you like in this directory
and the game will pick one at random on startup.

No music files are bundled with this repo -- they may be copyrighted by their
original composers.

## Where to find VGZ files

A large collection of freely available VGZ tunes can be found at:

  https://opl.wafflenet.com/

Download a few `.vgz` files and drop them into `tunes/`. That is all that is
needed -- the game loads them automatically.
